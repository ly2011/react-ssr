## 特性

1. use create-react-app

2. preloaded page data

3. code splitting

4. server-sid cookie support

## 安装

1. 安装依赖

```bash
yarn install
```

2. 客户端运行

```bash
yarn start
```

3. SSR模式运行(服务器端运行)

先执行客户端打包再运行server

```bash
yarn build && yarn serve
```
